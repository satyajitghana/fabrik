# Installation

## Quick Start

```bash
pnpm add @fabrik/ui zod motion
```

## With a specific LLM provider

```bash
# OpenAI
pnpm add @fabrik/ui zod motion openai

# Anthropic
pnpm add @fabrik/ui zod motion @anthropic-ai/sdk

# Google
pnpm add @fabrik/ui zod motion @google/genai
```

## Scaffold a new project

```bash
npx create-fabrik-app my-app
```

This creates a Next.js project with fabrik-ui pre-configured.

## Add to an existing project

```bash
npx fabrik init
```

This adds the configuration file and imports the styles.

## Import styles

Add to your root layout or global CSS:

```css
@import "@fabrik/ui/styles.css";
```

Or use a theme preset:

```css
@import "@fabrik/ui/themes/neutral.css";
```
