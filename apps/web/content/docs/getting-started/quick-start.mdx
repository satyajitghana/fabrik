# Quick Start

## 1. Create a component

```tsx
import { defineComponent } from "@fabrik/ui"
import { z } from "zod"

export const weatherCard = defineComponent({
  name: "weather_card",
  description: "Shows weather for a city",
  schema: z.object({
    city: z.string(),
    temp: z.number(),
    condition: z.enum(["sunny", "cloudy", "rainy", "snowy"]),
  }),
  component: WeatherCard,
})
```

## 2. Set up the provider

```tsx
import { Fabrik, Chat } from "@fabrik/ui/react"
import { openai } from "@fabrik/ui/openai"

export default function App() {
  return (
    <Fabrik
      provider={openai({ apiKey: "...", model: "gpt-4o" })}
      components={[weatherCard]}
    >
      <Chat />
    </Fabrik>
  )
}
```

## 3. That's it!

The LLM will automatically use your `weather_card` component when it decides to show weather data. It calls `show_weather_card({ city: "SF", temp: 68, condition: "sunny" })` and your React component renders with those props.

## Using the hook directly

```tsx
import { useChat, Message } from "@fabrik/ui/react"

function MyChat() {
  const { messages, input, send, isLoading } = useChat()

  return (
    <div>
      {messages.map(msg => <Message key={msg.id} message={msg} />)}
      <input value={input.value} onChange={e => input.set(e.target.value)} />
      <button onClick={send} disabled={isLoading}>Send</button>
    </div>
  )
}
```

## Server-side API key

Keep your API key safe on the server:

```tsx
// app/api/chat/route.ts
import { handler } from "@fabrik/ui/server"
import { openai } from "@fabrik/ui/openai"

export const POST = handler({
  provider: openai({ apiKey: process.env.OPENAI_KEY! }),
})
```

```tsx
// Client side
import { server } from "@fabrik/ui/server"

<Fabrik provider={server({ url: "/api/chat" })}>
```
